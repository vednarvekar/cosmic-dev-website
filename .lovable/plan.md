

# 🚀 Developer Portfolio - Space-Themed SPA

A stunning, dark-themed portfolio website for a Software Engineer with an immersive universe background, terminal-style hero, and interactive 3D elements.

---

## 🎨 Design & Theme

- **Dark mode** with deep space color palette (blacks, deep purples, cosmic blues)
- **Developer-centric colors** inspired by code editors/CLI: syntax green for success, error red, warning yellow, comment gray
- **Monospace fonts** (JetBrains Mono or Fira Code) for code sections, clean sans-serif for readability elsewhere
- **Moving universe background** using the uploaded video as a looping backdrop

---

## 📱 Sections (Single Page Application)

### 1. Hero Section - Terminal Intro
- Full-screen immersive entry with universe video background
- **Terminal-style introduction** with typing animation effect
- Shows your name and "Software Engineer" title appearing like commands being typed
- Animated cursor blink, retro terminal aesthetic
- Call-to-action to scroll or navigate

### 2. About Section - Code Block Style
- Personal introduction displayed inside a **styled code editor UI**
- Syntax highlighting with line numbers
- Information presented as code comments or variable declarations
- Smooth reveal animation on scroll

### 3. Tech Stack - Interactive Visual Display
- **Not boring text** - creative showcase of your skills
- Icons/badges for: React, TypeScript, Node.js, PostgreSQL
- Animated skill meters or orbiting tech icons in space theme
- Hover interactions with details
- Possibly styled as npm package dependencies or terminal output

### 4. Projects Section - Developer Cards
- 4-5 project showcases in **code-themed card design**
- Each card styled like a GitHub repo or terminal window
- Project title, description, tech stack tags
- Links to live demo/source code
- Hover effects with depth and glow matching space theme

### 5. Contact Section - Interactive Globe + Form
- Split layout: Contact form on one side, **3D interactive Earth** on the other
- Globe rotates and responds to cursor movement
- Form fields styled like terminal inputs
- **Resend email integration** for actual message delivery
- Success/error states with code-themed feedback

---

## ⚙️ Technical Features

- Smooth scroll navigation between sections
- Responsive design for all screen sizes
- Scroll-triggered animations throughout
- Performance optimized video background
- Three.js/React Three Fiber for 3D globe
- Edge function for contact form email delivery

---

## 🔧 Backend Integration

- **Lovable Cloud** for edge functions
- **Resend** for email delivery (requires API key setup)
- Contact form submissions sent to your email
